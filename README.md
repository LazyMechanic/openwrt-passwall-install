Script to install package `luci-app-passwall2`, `xray-core` and all dependencies

Single line install:
```
wget -O /tmp/passwall_install.sh https://github.com/LazyMechanic/openwrt-passwall-install/raw/refs/heads/master/install.sh && sh /tmp/passwall_install.sh --verbose
```
